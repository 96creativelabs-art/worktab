Installation Guide

Step 1: Download WorkTab

WorkTab is available as a Chrome extension. You can install it from the Chrome Web Store or load it as an unpacked extension for development.

Chrome Web Store (Recommended)

  1. Visit the Chrome Web Store
  2. Search for "WorkTab"
  3. Click "Add to Chrome"
  4. Confirm the installation

Step 2: First Launch

After installation, you'll see the WorkTab icon in your browser toolbar. Click it to open the extension.

Initial Setup

  • WorkTab will automatically detect your current tabs
  • Create your first workspace by clicking "New Workspace"
  • Organize your tabs by dragging them into workspaces
  • Customize settings in the navigation menu

Getting Started

Creating Your First Workspace

  1. Click the WorkTab icon in your browser toolbar
  2. Click "New Workspace" in the navigation bar
  3. Enter a name for your workspace (e.g., "Work Projects", "Research")
  4. Choose a color for your workspace
  5. Click "Create Workspace"

Adding Tabs to Workspaces

Method 1: Drag and Drop

Simply drag tabs from your browser into the workspace area.

Method 2: Right-Click Menu

Right-click on any tab or web page to access WorkTab options:

  • "Add to Workspace" - Choose an existing workspace
  • "Create New Workspace" - Create a new workspace with this tab
  • "Create rule for this site" - Auto-organize future tabs from this domain

Method 3: Keyboard Shortcuts

Use Alt+Shift+A to add the active tab or Alt+Shift+S to add all tabs to the last used workspace.

Frequently Asked Questions

General Questions

Is WorkTab free to use?

Yes, WorkTab is completely free to use with no limitations or premium features.

Does WorkTab work with other browsers?

Currently, WorkTab is only available for Chrome and Chromium-based browsers (like Edge, Brave, etc.).

Is my data private?

Yes, all your data stays on your device. WorkTab doesn't collect, store, or transmit any personal information.

Can I sync my workspaces across devices?

Currently, WorkTab stores data locally on your device. Cloud sync features may be added in future updates.

Technical Questions

How do I backup my workspaces?

Go to the Backup & Restore section in WorkTab settings. You can export your data as a JSON file and import it later.

Can I customize the auto-save interval?

Yes, you can change the auto-save interval in the Autosave settings. The default is 1 minute, but you can adjust it to your preference.

What happens if I uninstall WorkTab?

Your workspaces and settings will be lost unless you've backed them up. Make sure to export your data before uninstalling.

How do I use the right-click context menu?

Right-click on any tab or web page to see WorkTab options. You can add tabs to existing workspaces, create new workspaces, or set up automatic rules for specific websites.

Troubleshooting

Common Issues

WorkTab icon is not showing

If you can't see the WorkTab icon in your toolbar:

  • Check if the extension is enabled in chrome://extensions/
  • Try pinning the extension to your toolbar
  • Refresh your browser

Keyboard shortcuts not working

If keyboard shortcuts aren't responding:

  • Check if another extension is using the same shortcuts
  • Go to chrome://extensions/shortcuts to customize shortcuts
  • Make sure WorkTab has the necessary permissions

Workspaces not saving

If your workspaces aren't being saved:

  • Check your browser's storage permissions
  • Clear browser cache and try again
  • Disable and re-enable the extension
